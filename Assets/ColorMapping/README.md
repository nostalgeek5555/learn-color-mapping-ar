## Getting Started

ZIP 파일 다운로드후 알맞은 곳에 압축해제 후 유니티3d로 프로젝트 열기

1. Clone the repo or just download
```sh
git clone https://github.com/airar-dev/ColorMapping.git
```

2. AR 엔진 탑재

MaxstAR, EasyAR, Vuforia, ARkit, ARcore 중에 적용할 엔진 임포트
